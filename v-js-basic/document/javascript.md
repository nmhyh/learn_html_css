2. Toán tử trong Javascript
    - Toán tử số học - Arithmetic
        Ex: var a = 1 + 2;
    - Toán tử gán - Assignment
        Ex: var fullName = 'Ngu My Huynh';
    - Toán tử so sánh - Comparison
        Ex: var a = 1;
            var b = 2;
            if (a > b) {
                alert('Error');
            }
    - Toán tử logic - Logical (&&, ||, !)
        Ex: var a = 1;
            var b = 2;
            if (a > 0 && b > 0) {
                alert('a & b lon hon 0');
            }

    2.1. Toán tử số học
        +     - Cộng
        -     - Trừ
        *     - Nhân
        /     - Chia
        **    - Lũy thừa
        %     - Chia lấy số dư
        ++    - Tăng 1 giá trị số
        --    - Giảm 1 giá trị số

    2.2. Toán tử gán
    Toán tử         Ví dụ               Tương đương
    =               x = y               x = y
    +=              x += y              x = x + y
    -=              x -= y              x = x - y
    *=              x *= y              x = x * y
    /=              x /= y              x = x / y
    **=             x **= y             x = x ** y

*** Các giá trị sai: 0, false, '', "", undefined, NaN, null ***
